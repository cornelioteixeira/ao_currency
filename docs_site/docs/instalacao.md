# Instalação

Siga os passos abaixo para instalar e configurar a biblioteca no seu projeto.

## Requisitos

- Python 3.8 ou superior.

## Usando Pip

Para instalar a versão de teste (ou quando estiver no PyPI):

```bash
pip install ao_currency
```

## Instalação Local (Desenvolvimento)

Se você deseja contribuir ou testar localmente:

```bash
git clone https://github.com/cornelioteixeira/ao_currency.git
cd ao_currency
pip install -e .
```

## Verificando a Instalação

Você pode verificar se a CLI está funcionando:

```bash
ao-currency --help
```
