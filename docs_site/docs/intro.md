# Introdução

Bem-vindo à documentação oficial da **AO Currency Pro**, a biblioteca definitiva para manipulação de Moeda Angolana (Kwanza) em Python.

## O que é?

A `ao-currency-pro` é uma biblioteca de nível profissional desenhada para desenvolvedores que precisam lidar com o Kwanza (AOA) de forma robusta, segura e amigável.

## Principais Recursos

- **Sanitização Inteligente**: Limpa entradas de diversas fontes (Excel, PDF, strings sujas).
- **Formatação Bancária**: Segue rigorosamente os padrões de Angola (1.500,00 Kz).
- **Conversão por Extenso**: Traduz números para texto e vice-versa com suporte gramatical.
- **Integração Pydantic**: Tipagem forte e validação de dados nativa.
- **CLI Inclusa**: Ferramenta de linha de comando para uso rápido.

## Por que usar?

Diferente de soluções genéricas, esta lib foi construída focada no ecossistema de Angola, garantindo que `1.000,00` seja tratado corretamente como mil kwanzas, evitando erros comuns de precisão com o uso de `Decimal`.
